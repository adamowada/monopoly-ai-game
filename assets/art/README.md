# Game Art Assets

This directory contains local original art assets and prompt records governed by `ART_PLAN.md`.

## Directories

- `reference/`: AI-generated or hand-authored visual references that guide production artwork.
- `prompts/`: prompt records and review notes for generated art.
- `board/`: production board-square and board-center art.
- `cards/`: production property, Chance, Community Chest, and deck artwork.

Reference images are not automatically production assets. Before using a generated image in the app,
review it for unwanted logos, copied trade dress, text artifacts, poor small-size readability, and
accessibility needs.
